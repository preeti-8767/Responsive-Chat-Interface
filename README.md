# Responsive Chat Interface

A full-stack real-time chat application built using React.js, Node.js, Express, and Socket.io.  
This project demonstrates real-time WebSocket communication with a clean and responsive user interface.

---

# Features

- Real-time messaging using WebSockets
- Room-based chat system
- Message history support
- Auto-scroll to latest message
- Clean and responsive UI
- Modern component-based frontend structure
- Proper socket event cleanup (no duplicate messages)

---

##Tech Stack

# Frontend
- React.js
- CSS3
- Socket.io-client

# Backend
- Node.js
- Express.js
- Socket.io
- CORS
